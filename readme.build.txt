Информация
==========

В этой папке находится "сборка" приложения `iridoscope`.

https://iridoscope.github.io
https://github.com/iridoscope/iridoscope

Хостинг
=======

Данную "сборку" следует "захостить" в интернете в качестве "статического" веб-сайта где-нибудь на сервере или в облаке.

Настройки
=========

Для редактирования настроек, откройте файл `index.html` и найдите там следующие строчки вверху:

```
<script>
	// Set your config parameters here.
	// See the readme for more details.
	var CONFIG = {}
</script>
```

Внутри этих фигурных скобок можно добавлять собственные настройки. Смотрите документацию проекта, чтобы увидеть список возможных настроек.
